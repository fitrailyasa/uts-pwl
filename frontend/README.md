## Tech Stack

- React
- TypeScript
- Vite
- Bootstrap
- useSWR
- React Router DOM
- React Hook Form

## How to run

- Clone this repository

```bash
git clone https://github.com/fitrailyasa/uts-pwl.git
```

- change directory

```bash
cd frontend/utspwl
```

- Install dependencies

```bash
npm install
```

- Run the app

```bash
npm run dev
```
